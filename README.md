# <hz-range>

Custom element range component, a part of Bigfoot UI kit.

## Installation

``` npm install bigfoot-ui ```

## Usage

1. full library with polyfills

index.js

```
import * as bigfootui from "bigfoot-ui";
```

index.html

```
<html lang="en">
    <body>
        <hz-range></hz-range>
        <script src="index.js"></script>
    </body>
</html>
```
